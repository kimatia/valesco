Download and Install fpdf class from http://www.fpdf.org/
Keep a copy of fpdf.php file in this directory,
Keep the font directory inside this directory. 

Use the SQL_dump.txt file to create student tables ( student , student3 )  in your MySQL database
Open config.php file to enter your MySQL login details. 
Open index.php file to see the records in your borwser. 
Open index-pdf.php file to generate PDF document. 

index1.pdf file will list all students from student3 tables. Click the mark column to display mark sheet of individual student.
